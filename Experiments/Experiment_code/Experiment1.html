This Experiment was used as a layout for making my homepage on 20th January 2014.

I implemented this experiment to clear my basics about css and html.

In this experiment I explored the css propreties like :

  • Absolute:

    It specifies the positioning method for an element. I was having trouble with the layout and absolute solved those problems.

  • Margin:

    It clears an area around the element. It helps element to avoid overlapping each other.

  • Class and Id:

    Understood how Class and id selectors work. They help to give a particular class or a particular element custom modification.
