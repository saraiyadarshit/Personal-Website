Transition of Images using Animation

Documentation :

The purpose of this experiment is to learn how to make transitive images on a website only using css. This will enable us to make more interactive websites.

For the first transition I have used the opacity property in the css to create the effect of transition. The images are placed on top of each other and when the mouse cursor hovers over the image the opacity of the top image reduces to zero.

For the second transition I have used the two properties opacity and rotate of the css to create the transition effect. I have added the rotation effect on the image to make the transition more interesting.

For the third traansition I have used Animation property of the css to create transition. I applied the animation property on the images and the iteration-count property to infinity so that the images keep transitioning.
