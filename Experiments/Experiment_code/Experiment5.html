Documentation

The purpose of this experiment was to learn Validation Using Javascript.

The User will enter values in the given fields to register. Upon clicking the submit button a Javascript will run to validate the inputs given by the user. If there is an invalid input, the Javascript will throw an alert with information to rectify the mistake.

There is also a function to check the strength of the password which uses JavaScript and Regular Expression to evaluate the strength of the Password.

The Email Id field has input type "email", a new feature in HTML5 which automatically verifies the input.

This Experiment will be used in the project to create new member registration page.

Used HTML5, CSS3 and Javascript to implement this experiment.
