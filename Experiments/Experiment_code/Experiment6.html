The purpose of this experiment was to create a feedback form which is very important for user inputs and suggestions.

The User will enter the comments in the Comments text area and the response will be sent to my personal email ID.

This will help me to analyze user inputs and make changes on my website to meet their expectations.

Upon clicking the send email button a Javascript will run to validate the inputs given by the user. If there is an invalid input, the Javascript will throw an alert with information to rectify the mistake.

The Email procedure is not working directly right now.

This Experiment will be used in the project to create the feedback page.

Used Mozilla thunderbird, HTML5, CSS3 and Javascript to implement this experiment.
